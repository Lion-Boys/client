## Client

### Onboarding

#### 0. Prerequisites

```bash
npm install --global corepack@latest
corepack enable pnpm
```

#### 1. Install Dependencies
```bash
pnpm install
```
- Add a package
    ```bash
    pnpm add <pkg>
    ```
- Add a dev dependency
    ```bash
    pnpm add -D <pkg>
    ```

#### 2. Run Dev Server
```bash
pnpm dev
```

#### 3. Build
```bash
pnpm build
```

#### 4. Production
```bash
pnpm preview
```
